Kotton.css

한글이 잘 어울리는 스타일시트

한글 문서에 적합한 HTML 5 Classless 스타일시트

한글을 위한 문서 스타일시트

많은 CSS 프레임워크와 스타일시트들은 영어문서를 기준으로 작성되었습니다.

그렇기 때문에 한글 문서에 적용한 뒤 생각과는 다른 외적인 모습에 당황하게 됩니다.

Kotton.css는 처음부터 한글을 기본으로 스타일이 작성되어 있습니다.

Kotton.css를 통해서 예상 가능한 한글 문서 스타일을 적용해보세요.

웹표준 스펙에 맞추어 작성

Kotton.css는 Class 사용을 최소화한 HTML5 시멘틱 마크업을 지향합니다. Header와 Article을 포함하여 HTML5 마크업 표준을 준수하여 작성하는 것만으로 훌륭한 한글 문서가 탄생합니다.

쉬운 적용 및 해제

Kotton.css는 적용 범위를 자유롭게 지정할 수 있습니다. 만약 원한다면 문서의 일부분만 적용하거나, 문서의 일부분만 적용하지 않을 수 있습니다. Kotton.css는 클래스를 최소한으로 사용하기 때문에, 언제든지 쉽게 다른 스타일시트로 변경이 가능합니다. 프로토타입용으로 너무 좋아요

유연한 라이선스

Kotton.css는 개인*기업 모두 자유롭게 사용 할 수 있으며, 자유로운 라이선스들의 모임으로 탄생하였습니다.

사용한 외부 폰트(본고딕, 나눔명조)와 tablericons 역시 자유로운 라이선스를 사용합니다

(상세 라이선스 조항는 각 폰트와 아이콘을 확인하세요)
